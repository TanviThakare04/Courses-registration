# Course Registration System

##  Project Description
This is a full stack web application where students can register and enroll in different courses.

## Technologies Used
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MySQL

## Project Structure
- frontend → User Interface
- backend → Server and APIs
- database.sql → Database structure

##  Features
- Student Registration & Login
- View Available Courses
- Register for Courses
- Store Data in MySQL Database

##  Database
- students table → stores student details
- courses table → stores course list
- registrations table → manages student-course relationship

##  How to Run
1. Install Node.js
2. Start backend server
3. Start frontend
4. Import database.sql in MySQL
